# CODE OF CONDUCT
## OUR COMMITMENT  
We are committed to fostering a welcoming, respectful, and collaborative environment for everyone participating in the MiniLang project. All contributors are expected to communicate professionally and work together to improve the project.
***
## **Expected Behavior**  
Contributors are expected to:  
- Be respectful and considerate towards all participants.
- Use clear and professional language in discussions and documentation.
- Provide constructive feedback during code reviews.
- Follow the project's coding standards and contribution guidelines.
- Report bugs and issues responsibly with sufficient details.
- Respect different opinions and collaborate effectively.
- Help maintain a positive and inclusive development environment.
***
## **Unacceptable Behavior**  
The following behaviors are unacceptable:  
- Harassment, discrimination, or offensive language.
- Personal attacks or disrespectful communication.
- Intentional disruption of discussions or project activities.
- Submitting plagiarized or malicious code.
- Sharing confidential or private information without permission.
- Any behavior that negatively affects the project's collaborative environment.
- Contribution Guidelines  
***
## **When contributing to MiniLang:**  
- Follow the instructions provided in CONTRIBUTING.md.
- Create meaningful commit messages.
- Test your changes before submitting them.
- Ensure that your code does not introduce compilation errors.
- Keep documentation updated whenever new features are added.
- Submit changes through pull requests whenever possible.
- Enforcement  
Project maintainers reserve the right to review contributions and take appropriate action if this Code of Conduct is violated. This may include requesting changes, rejecting contributions, or restricting participation in the project.
***  
## **Contact** 
For questions regarding this Code of Conduct or the project, please contact the project maintainers through the GitHub repository or the contact information provided in the project documentation.  
By participating in this project, you agree to follow this Code of Conduct and contribute toward maintaining a professional, respectful, and collaborative development environment.
