# INSTALLATION & USAGE INSTRUCTIONS
## INSTALLATION  
**Prerequisites**
- C++ Compiler (GCC/G++ 11 or later recommended)
- Git
- Visual Studio Code / Code::Blocks / Any C++ IDE  
**Clone the Repository**  
- git clone https://github.com/<RAGHAV4056>/MiniLang.git
- cd MiniLang  
**Run the Interpreter**  
- ./MiniLang  
(On Windows)  
- MiniLang.exe
***
## USAGE INSTRUCTIONS
- Launch the MiniLang interpreter after successful compilation.
- The interpreter will display a command prompt, indicating that it is ready to accept user input.
- Enter MiniLang commands one at a time using the supported command syntax.
- The interpreter processes each command by performing lexical analysis, syntax validation, and execution.
- Results of successful operations are displayed immediately on the console. If an invalid command or runtime error is encountered, an appropriate error message is shown.
- Continue entering commands until all desired operations have been completed.
- To terminate the interpreter session, enter the following command: exit
***
## USAGE  
Launch the interpreter and enter commands interactively. Example of Some Possible Commands:-  
**Store a Variable**  
store age as 20  
**Display a Variable**  
display variable age  
**Update a Variable**  
update age as 25  
**Arithmetic Operation**  
add num1 and num2 into result  
**Compare Variables**  
compare num1 & num2  
**Increase or Decrease**  
increase age by 5  
decrease age by 2  
**Show All Variables**  
showvars  
**Exit the Interpreter**  
exit
