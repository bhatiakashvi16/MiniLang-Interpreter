## PROJECT INFORMATION  
MiniLang is a lightweight command-based programming language and interpreter developed in C++ for educational purposes. The project demonstrates the core concepts behind programming language design, including lexical analysis, parsing, command interpretation, variable management, and runtime execution. It supports multiple data types, arithmetic operations, string manipulation, comparison operations, and exception handling while following object-oriented programming principles such as abstraction, inheritance, and polymorphism.  
The project is developed following open-source software development practices using Git and GitHub, emphasizing version control, collaborative development, documentation, and structured project management.

## INSTALLATION  
**Prerequisites**
- C++ Compiler (GCC/G++ 11 or later recommended)
- Git
- Visual Studio Code / Code::Blocks / Any C++ IDE  
**Clone the Repository**  
- git clone https://github.com/RAGHAV4056/MiniLang-Interpreter.git
- cd MiniLang  
**Run the Interpreter**  
- ./MiniLang  
(On Windows)  
- MiniLang.exe

## USAGE  
Launch the interpreter and enter commands interactively. Example of Some Possible Commands:-  
**Store a Variable**  
store age as 20  
**Display a Variable**  
display variable age  
**Update a Variable**  
update age as 25  
**Arithmetic Operation**  
add num1 and num2 into result  
**Compare Variables**  
compare num1 & num2  
**Increase or Decrease**  
increase age by 5  
decrease age by 2  
**Show All Variables**  
showvars  
**Exit the Interpreter**  
exit
