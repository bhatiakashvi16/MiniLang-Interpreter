# CHANGELOG
All notable changes to MiniLang will be documented in this file.
The project follows semantic versioning:  
MAJOR.  
MINOR.  
PATCH.
- MAJOR version for incompatible or significant feature additions.
- MINOR version for backward-compatible feature additions.
- PATCH version for bug fixes and minor improvements.
---
# [1.0.0] - INITITAL RELEASE
## ADDED
- Variable creation using `store`.
- Variable display using `display`.
- Variable update using `update`.
- Variable deletion using `delete`.
- Variable comparison using `compare`.
- Variable increment using `increase`.
- Variable decrement using `decrease`.
- Arithmetic operations:
  - add
  - subtract
  - multiply
  - divide
- Queue-based command execution.
- Tokenizer for command parsing.
- Runtime exception handling.
- Support for multiple datatypes:
  - Integer
  - Float
  - Boolean
  - String
- Show all variables using `showvars`.
## FEATURES
- Object-Oriented Design.
- Command Pattern implementation.
- Modular interpreter architecture.
- Interactive command-line interface.
